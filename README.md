# Junior Developer Landing Page

A simple portfolio-style landing page to showcase skills, experience, and contact information for a junior software developer.

## Files

- `index.html` – landing page markup
- `styles.css` – responsive styling and layout
- `script.js` – project card rendering

## Usage

Open `index.html` in a browser or use a local server to preview the page.

## Customize

- Update the hero text and contact email in `index.html`
- Change the skills list and project details in `script.js`
- Adjust colors and spacing in `styles.css`
